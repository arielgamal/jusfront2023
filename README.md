## Informações
Utilizei Ubuntu para realizar esse projeto.

Inicie esse passos após terminar de rodar a API.

## Começando:
Entre na pasta jusfront
```bash
cd jusfront
```

## Comando de instalação
```bash
npm install
```

## Iniciando a aplicação

```bash
npm run build
npm start
```

## Iniciando a aplicação
Vá no navegador e digite:

```
http://localhost:3000/
```
