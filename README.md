## Informações
Utilizei Ubuntu para realizar esse projeto e node v21.1.0

Siga esses passos para iniciar a aplicação frontend.

## Começando:
Entre na pasta jusfront
```bash
cd jusfront2023
```

## Comando de instalação
```bash
npm install
```

## Iniciando a aplicação

```bash
npm run build
npm start
```

## Iniciando a aplicação
Vá no navegador e digite:

```
http://localhost:3000/
```
